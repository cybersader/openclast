# Clasty Logo Generation Prompts

Brand identity derived from "pyroclast" — a fragment of volcanic rock ejected from an eruption. The visual language is: obsidian-black angular shard with magma-orange glowing cracks.

## Color Palette (matches clasty-theme.css)

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Void | True Black | `#0c0c0c` | Deepest background |
| Stone Dark | Dark Stone | `#1c1917` | Nav, sidebar bg |
| Stone Mid | Medium Stone | `#292524` | Surfaces, code blocks |
| Magma | Deep Red-Orange | `#dc4a1a` | Primary accent, glowing cracks |
| Magma Deep | Dark Red | `#b83a10` | Accent on light backgrounds |
| Ember | Light Orange | `#ef6c35` | Accent highlights, hover states |
| Ash White | Warm White | `#e7e5e4` | Text on dark backgrounds |

## Style Notes for All Prompts

- Add "vector style, flat design, clean edges" for crisp results
- Add "no gradients" if you want pure flat, or "subtle radial glow only" for the ember effect
- Add "SVG-ready, suitable for scaling" for icons
- For AI image generators (Midjourney, DALL-E, Ideogram): append "--style raw" or "graphic design, logo design, brand identity" to get cleaner results
- For mascot versions: append "kawaii style, chibi, cute character design, mascot illustration"
- Avoid: "realistic, 3D, photorealistic, detailed texture" — keep it stylized

## Version 8: Recraft Workflow Guide

Step-by-step for generating the full logo set in Recraft:

### Step 1: Nail the silhouette
Generate the **flat color** version (Treatment 2) first. This establishes the canonical shape. Iterate until the shard silhouette feels right — it should be recognizable at 32px and distinctive at 512px.

### Step 2: Crystal hero version
Take the same silhouette and generate the **crystal/gem** version (Treatment 1). Use Recraft's "Realistic" or "Digital Art" style. Reference: the obsidian-daily-notes-ng crystal calendar logo.

### Step 3: Derive all mono versions
From the flat version, generate **mono white**, **mono black**, **mono color**, and **outlined** variants. These should be trivial once the silhouette is locked.

### Step 4: Export as SVG
