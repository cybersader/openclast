# Clasty Logo Generation Prompts

Brand identity derived from "pyroclast" — a fragment of volcanic rock ejected from an eruption. The visual language is: obsidian-black angular shard with magma-orange glowing cracks.

## Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Primary | Obsidian Black | `#1a1a2e` | Main shard body, dark backgrounds |
| Accent | Magma Orange | `#e85d26` | Glowing cracks, highlights, CTAs |
| Glow | Amber | `#f59e0b` | Secondary glow, warmth |
| Deep Black | Void | `#0f0f17` | Backgrounds, dark mode |
| Light | Ash White | `#e8e6e3` | Text on dark, light mode backgrounds |
| Cool Accent | Volcanic Blue | `#3b82f6` | Optional secondary, links, info states |

## Version 1: Primary Logo Mark (Icon Only)

### Dark Background Version
```
A single angular geometric shard shape — irregular pentagon or hexagonal fragment — in obsidian black (#1a1a2e) with two or three thin magma-orange (#e85d26) crack lines running through it, glowing subtly with amber (#f59e0b) edges. The shard is slightly tilted, asymmetric, like it was just broken off. Minimal, flat design. No text. Dark charcoal background (#0f0f17). Clean vector style suitable for a software product icon. No gradients except the subtle glow on crack edges.
```

### Light Background Version
```
A single angular geometric shard shape — irregular pentagon or hexagonal fragment — in obsidian black (#1a1a2e) with two or three thin magma-orange (#e85d26) crack lines running through it, glowing subtly. The shard is slightly tilted, asymmetric, like a volcanic rock fragment. Minimal, flat design. No text. Clean white (#ffffff) background. Clean vector style suitable for a software product icon. The shard should have enough contrast to stand alone on white.
```

The shard is slightly tilted, asymmetric. Minimal, flat, single-color design. No text. White background. Suitable for single-color printing, embossing, or watermarks.

```
A single angular geometric shard shape — irregular pentagon or hexagonal fragment — in solid white with two or three thin dark crack lines running through it. The shard is slightly tilted, asymmetric. Minimal, flat, single-color design. No text. Transparent or dark background. Suitable for watermarks on dark surfaces, white-on-dark headers.
```

## Version 3: Mascot (Clasty the Shard)

### Full Character
```
A cute cartoon mascot character that is a small angular chunk of obsidian-black volcanic rock. The rock is roughly pentagonal/hexagonal, slightly asymmetric with jagged edges — like a volcanic glass fragment. Two or three thin cracks run through the body, glowing magma-orange (#e85d26) and amber (#f59e0b). The character has two simple round white eyes with small dark pupils, placed on the upper face of the shard. A tiny wisp of steam or ember particles drift upward from the top. The character is standing on a subtle shadow. Friendly, approachable expression. Kawaii/chibi art style similar to GitHub's Octocat or Astro's Houston mascot. White or transparent background.
```

### Mascot Waving / Greeting
```
Same volcanic rock shard mascot character as above, but with a small angular arm (made of the same dark rock material) raised in a friendly wave. The magma cracks glow slightly brighter as if energized. Happy, welcoming expression. A few tiny ember particles floating around. Suitable for documentation landing pages, onboarding screens, 404 pages.
```

### Mascot Splitting and Merging (CRDT Sync Metaphor)
```
Two smaller versions of the volcanic shard mascot character, positioned side by side. Between them, glowing magma-orange connection lines or particles flowing in both directions — representing real-time sync. Both shards have happy expressions. The left shard has a tiny document icon floating above it, the right shard has the same document materializing. Dark background with subtle glow. Represents the CRDT collaboration feature.
```

### Mascot Guarding a Gate (RBAC Metaphor)
```
The volcanic shard mascot character standing in front of a simple geometric gate or doorway made of dark stone. The gate has the same magma-orange glow in its seams. The mascot has a determined but friendly expression, one arm extended in a "stop/check" gesture. A small glowing key or shield floats nearby. Represents the access control / mount orchestration feature.
```

## Style Notes for All Prompts

- Add "vector style, flat design, clean edges" for crisp results
- Add "no gradients" if you want pure flat, or "subtle radial glow only" for the ember effect
- Add "SVG-ready, suitable for scaling" for icons
- For AI image generators (Midjourney, DALL-E, Ideogram): append "--style raw" or "graphic design, logo design, brand identity" to get cleaner results
- For mascot versions: append "kawaii style, chibi, cute character design, mascot illustration"
- Avoid: "realistic, 3D, photorealistic, detailed texture" — keep it stylized
