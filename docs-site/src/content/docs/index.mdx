## Three-Layer Architecture

<CardGrid>
  <Card title="VNC Gateway" icon="laptop">
    Stream Obsidian desktop to the browser via Kasm Workspaces. Full desktop experience, zero installs.
  </Card>
  <Card title="CRDT Sync" icon="random">
    Conflict-free real-time collaboration powered by Yjs. Multiple users edit the same vault simultaneously.
  </Card>
  <Card title="Mount Orchestration" icon="setting">
    Role-based access control at the folder level. Each user sees only what they're authorized to access, assembled as one seamless vault.
  </Card>
  <Card title="Enterprise Ready" icon="approve-check">
    SSO integration, audit logging, DLP controls, and container hardening for regulated environments.
  </Card>
</CardGrid>
