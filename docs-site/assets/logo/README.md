# OpenClast Logo Assets

Volcanic obsidian shard — the visual identity of OpenClast (derived from "pyroclast"). The mascot character is named **Clasty**.

## Treatment System

All variants share the same base silhouette. See `../LOGO_PROMPTS.md` for generation prompts.

| Treatment | File Pattern | Usage |
|-----------|-------------|-------|
| Crystal/gem textured | `logo-recraft-crystal.svg` | Hero images, splash, marketing |
| Flat color | `logo-recraft-flat.svg` | Primary logo, favicon source |
| Detailed flat | `logo-recraft-detailed.svg` | Mid-size contexts |
| Mono white (currentColor) | `logo-recraft-currentcolor.svg` | Dark backgrounds, overlays |
| Mono black | `logo-recraft-dark.svg` | Light backgrounds, print |
| Mono magma | `logo-recraft-magma.svg` | Brand color fill |
| Outlined / badge | `logo-recraft-outlined.svg` | Stickers, patches, badges |
| Dark purple variant | `logo-recraft-dark-purple.svg` | Obsidian ecosystem contexts |

## Generation Tool

Logos generated with [Recraft](https://www.recraft.ai/). Original AI outputs stored in `sources/`.
